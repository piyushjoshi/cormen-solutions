Exercise 12.3-3

Solution

Let $h_i$ be the height of the tree when i'th node is being inserted.

Time taken to insert i'th node:

$T_i$ = O($h_i$)

Then running time of building the tree will be:

$T(n)$ = $\sum\limits_{i=1}^n T_i$ = $\sum\limits_{i=1}^n O(h_i)$ = $O(\sum\limits_{i=1}^n h_i)$

Worst case occurs when input numbers are already sorted. In that case:

$h_i$ = $(i-1)$
thus: $T(n)$ = $O(\sum\limits_{i=1}^n (i-1))$ = $O(n^2)$

Best case occurs when the resulting tree is balanced. In that case:
$h_i$ = $\lfloor log(i) \rfloor$
thus: $T(n)$ = $O(\sum\limits_{i=1}^n (\lfloor log(i) \rfloor) )$ = $O(n log(n))$
